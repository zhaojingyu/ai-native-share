# AI Native Collaboration Share

Static Vercel-ready page for sharing an AI Native organization collaboration model.

## Local preview

```bash
npm run dev
```

Open `http://127.0.0.1:4173`.

The page also works by opening `index.html` directly in a browser.

## Deploy

From this directory:

```bash
npx vercel --prod
```

The local machine still needs Vercel login or `VERCEL_TOKEN`.
